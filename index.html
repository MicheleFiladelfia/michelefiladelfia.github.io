About

Hi, my name is Michele Filadelfia, i'm a self-taught

years old Italian software developer with a significant passion for computing science, i began my studies by enrolling in a technical and information technology high school where I learned Arduino basics, HTML and CSS, Java basics, networking concepts and other technologies.

Proceeding with the studies I realized that the school was not giving me enough knowledge in programming, so I decided to study the C programming language from self-taught with a book called "C Programming: A Modern Approach" by Kim N. King to improve my coding thinking.

After learned to code with the programming language C, I decided to study some technologies related to the Back-End, so I chose firstly to improve my skills in the Front-End(HTML, CSS and JS) and then to learn Node.JS with the Express framework for the Back-End.

Later I started learning algorithms and data structures to improve my problem solving and programming skills from various books such as "Introduction to Algorithms (CRLS)", "Guide to competitive programming (Antti Laaksonen)" and others.

While studying algorithms and data structures, I read books regarding clean code, software engineering, agile methodology, extreme programming etc. from books such as "Clean Code (Robert C. Martin)", "The pragmatic programmer (Dave Thomas)" and others.
